# React Native Bottom Drawer

An example of the implementation of a lower drawer, with the ability to open and close both by pressing a button and by swiping a finger up and down

# Getting Started

## Install modules:

```npm install```

### Install for iOS:

```
cd ios
pod install
```

## Start

```bash
# using npm
npm start

# OR using Yarn
yarn start
```
